隐私政策

最后更新日期：2026 年 4 月 12 日

重要提示：本扩展严格遵守 Chrome Web Store 用户数据隐私政策。我们仅收集实现核心功能所必需的最少数据，并将所有数据存储在用户本地设备或用户自己的后端服务器中，不会向任何第三方共享或出售用户数据。

一、我们收集的数据

为了提供墨问心情分析功能，本扩展会收集和处理以下数据：

1.1 墨问笔记数据

  • 笔记 UUID：用于唯一标识每篇笔记，实现笔记同步和去重
  • 笔记标题：用于在心情趋势图和笔记列表中显示
  • 笔记内容：用于 AI 心情分析，分析完成后不会永久存储原始内容
  • 发布时间：用于按时间排序和趋势分析

1.2 认证信息

  • Cookie：用于访问墨问笔记 API 获取您的笔记数据。Cookie 仅在需要时通过 background script 临时使用，不会存储到扩展的本地存储中

1.3 心情分析数据

通过分析笔记内容，本扩展会生成以下衍生数据：

  • 心情分数 (mood_score)：数值范围 -10 到 40，表示心情温度
  • 心情标签 (mood_label)：如"愉悦"、"平静"、"焦虑"等文字描述
  • 天气类型 (weather_type)：根据心情分数映射的天气图标（晴、雨、阴等）
  • 分析原因 (reason)：AI 分析心情的简要原因说明
  • 分析时间 (created_at)：记录心情分析的时间戳

二、数据收集方式

本扩展通过以下方式收集数据：

  • Chrome Content Scripts：在 note.mowen.cn 域名下运行，用于检测当前页面是否为笔记详情页
  • Chrome Cookies API：读取 note.mowen.cn 的认证 Cookie，用于访问墨问 API
  • Chrome Tabs API：获取当前活动标签页的 URL，判断是否在笔记详情页
  • 墨问 API 接口：通过 HTTPS 请求获取笔记列表和笔记详情
  • 后端 API 接口：将笔记内容发送至您的后端服务器进行 AI 心情分析

三、数据使用目的

收集的数据仅用于以下目的：

  • 心情分析：使用 AI 分析笔记内容，生成心情分数和标签
  • 趋势可视化：在侧边栏中以温度曲线和天气图标展示心情变化趋势
  • 笔记列表展示：显示最近分析的笔记及其心情分数
  • 笔记同步：定期同步新笔记并进行心情分析，避免重复分析
  • 单篇笔记分析：当您在笔记详情页时，提供"共享心情"功能分析当前笔记

四、数据存储位置

本扩展的数据存储在以下位置：

4.1 本地存储（用户设备）

  • Chrome Storage API：不存储任何用户数据，仅可能存储扩展配置信息（如 API 地址）
  • 浏览器内存：运行时临时数据（如当前笔记 UUID、分析结果），关闭侧边栏后自动清除

4.2 后端服务器存储

心情分析结果存储在用户自己部署的后端服务器中（默认地址：mowen-mood.yuanbixue.com 或用户自定义地址）：

  • SQLite 数据库：存储笔记 UUID、标题、心情分数、心情标签、天气类型、分析原因、分析时间
  • 不存储：笔记完整内容、用户个人信息、认证 Cookie、密码等敏感信息
  • 服务器位置：由用户自行部署和控制，开发者无法访问

五、数据共享与披露

5.1 不共享的数据

本扩展不会向以下方共享任何用户数据：

  • ❌ 广告商或营销公司
  • ❌ 数据经纪商或聚合服务商
  • ❌ 社交媒体平台
  • ❌ 信用报告机构
  • ❌ 任何未经用户明确同意的第三方

5.2 必要的 API 通信

为实现核心功能，数据会在以下端点之间传输：

  • note.mowen.cn：获取笔记数据（使用您的认证 Cookie）
  • 后端服务器（mowen-mood.yuanbixue.com 或自定义地址）：发送笔记内容进行心情分析，接收分析结果
  • 所有通信均使用 HTTPS 加密传输

5.3 法律要求

仅在以下情况下可能披露数据：

  • 遵守法律法规或法院命令
  • 保护本扩展用户、公众或系统的权利、财产或安全

六、权限使用说明

本扩展请求以下 Chrome 权限，每个权限都为实现核心功能所必需：

权限名称 使用目的
cookies 读取 note.mowen.cn 的认证 Cookie，用于访问墨问 API 获取笔记数据
sidePanel 在浏览器侧边栏中展示心情分析界面
host_permissions: note.mowen.cn 注入 content script 检测笔记详情页，调用墨问 API
host_permissions: 后端服务器 与后端服务器通信，进行心情分析和数据同步

七、数据安全

我们采取以下措施保护您的数据安全：

  • HTTPS 加密：所有网络通信均使用 HTTPS 协议加密
  • 最小化存储：后端数据库仅存储分析结果（心情分数、标签等），不存储笔记完整内容
  • 本地优先：运行时数据仅保存在浏览器内存中，关闭侧边栏后自动清除
  • 无持久化认证：Cookie 仅临时使用，不存储到本地
  • 用户控制：后端服务器由用户自行部署，完全掌控数据

八、用户权利

您对自己的数据享有以下权利：

  • 访问权：您可以在扩展侧边栏查看所有已分析的心情数据
  • 删除权：您可以清除扩展数据（Chrome 设置 > 扩展 > 墨问心情 > 清除数据）
  • 停止收集：您可以禁用或卸载扩展，立即停止所有数据收集
  • 服务器数据管理：您拥有后端服务器的完全控制权，可以随时删除数据库

九、儿童隐私

本扩展不面向 13 岁以下儿童，也不会故意收集儿童的个人信息。如果我们发现无意中收集了儿童数据，将立即删除。

十、政策更新

我们可能会不时更新本隐私政策。更新后的政策将：

  • 在此页面发布最新版本，并更新日期
  • 重大变更时通过 Chrome Web Store 更新说明通知用户
  • 继续使用本扩展即表示您同意更新后的隐私政策
