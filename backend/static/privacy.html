隐私政策

最后更新日期：2026 年 4 月 11 日

概述

本隐私政策说明了「墨问心情」（以下简称"本扩展"）如何处理用户数据。我们重视用户的隐私，并致力于保护您的个人信息安全。

数据收集声明

本扩展不收集、存储、传输或分享任何个人或敏感用户数据。

具体而言，本扩展：

  • 不收集个人身份信息（如姓名、邮箱、地址、电话号码等）
  • 不收集浏览历史或网页浏览活动数据
  • 不访问或存储用户的认证信息（如密码、Cookie 等）
  • 不收集财务或支付信息
  • 不收集健康信息
  • 不收集用户生成的内容或个人通讯
  • 不向任何第三方传输或分享用户数据

本地存储

本扩展可能在本地使用 Chrome Storage API 存储必要的配置信息，以实现核心功能。这些数据：

  • 仅存储在用户本地设备上
  • 不会传输到任何外部服务器
  • 用户可随时通过清除扩展数据来删除

权限使用说明

本扩展仅请求实现其功能所必需的最小权限。所有权限均用于提供核心功能，不会用于收集用户数据。

政策合规

本扩展严格遵守以下政策：

  • Chrome Web Store 用户数据政策 (User Data Policy)
  • Chrome Web Store 开发者计划政策
  • 有限使用要求 (Limited Use Requirements)
